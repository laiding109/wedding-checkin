1
建立 Google Sheets 試算表
開啟 sheets.google.com 新增一份空白試算表，建立兩個分頁：
來賓名單（欄位：姓名、群組、桌號、人數、報到時間、狀態）

2
貼上 Apps Script 程式碼
在試算表點選「擴充功能 → Apps Script」，刪除原有內容，貼上以下程式碼，再點「部署 → 新增部署 → 網路應用程式」，執行身分選自己，存取權選所有人，取得部署網址。
function doPost(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const data = JSON.parse(e.postData.contents);
  const sheet = ss.getSheetByName(data.type);
  if (sheet) sheet.appendRow(data.row);
  return ContentService.createTextOutput('ok');
}
function doGet(e) {
  return ContentService.createTextOutput('ok');
}
