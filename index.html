步驟 1：建立 Google Sheets
一次設定，永久適用
API 網址會存在 Sheets 的設定頁，任何裝置、任何瀏覽器打開這個網頁都會自動讀取，不需要重複設定。
  1. 前往 sheets.google.com 建立空白試算表
  2. 名稱建議取「婚禮報到系統」
步驟 2：貼上 Apps Script 程式碼
這段程式是資料讀寫的橋梁，getDisplayValues() 確保時間格式正確，不會變成奇怪日期格式。
  1. 點「擴充功能 → Apps Script」
  2. 刪除所有內容，貼上以下程式碼
  3. 按「儲存」
function doPost(e) { try { const d = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); if (d.action === 'save') { let sh = ss.getSheetByName(d.sheet); if (!sh) sh = ss.insertSheet(d.sheet); sh.clearContents(); if (d.rows && d.rows.length) sh.getRange(1,1,d.rows.length,d.rows[0].length).setValues(d.rows); } return out({ok:true}); } catch(err) { return out({ok:false,e:err.message}); } } function doGet(e) { try { const sh = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName(e.parameter.sheet); const rows = sh ? sh.getDataRange().getDisplayValues() : []; return out({ok:true,rows}); } catch(err) { return out({ok:false,e:err.message}); } } function out(o) { return ContentService.createTextOutput(JSON.stringify(o)) .setMimeType(ContentService.MimeType.JSON); }
步驟 3：部署並貼上網址
  1. 點「部署 → 新增部署 → 網路應用程式」
  2. 執行身分「我」，存取權「所有人」
  3. 按「部署」，複製網址貼在下方
