步驟 1：建立 Google Sheets
為什麼用 Google Sheets？
資料存在雲端，不管換電腦、換瀏覽器、多台手機同時操作，都能即時同步。完全免費，只需要 Google 帳號。
  1. 前往 sheets.google.com，建立一份新的空白試算表
  2. 試算表名稱可以取「婚禮報到系統」
  3. 建立完成後複製上方的網址（待用）
步驟 2：設定 Apps Script
在試算表裡建立一個小程式，讓這個網頁可以讀寫你的 Sheets 資料。
  1. 在試算表點選上方選單「擴充功能 → Apps Script」
  2. 刪除編輯器內所有原有內容
  3. 將以下程式碼全部貼上並按「儲存」
const SHEET_ID = SpreadsheetApp.getActiveSpreadsheet().getId(); function doPost(e) { try { const d = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); if (d.action === 'save') { let sh = ss.getSheetByName(d.sheet); if (!sh) sh = ss.insertSheet(d.sheet); sh.clearContents(); if (d.rows && d.rows.length) sh.getRange(1,1,d.rows.length,d.rows[0].length).setValues(d.rows); } else if (d.action === 'read') { const sh = ss.getSheetByName(d.sheet); const rows = sh ? sh.getDataRange().getValues() : []; return ContentService.createTextOutput(JSON.stringify({ok:true,rows})).setMimeType(ContentService.MimeType.JSON); } return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService.createTextOutput(JSON.stringify({ok:false,error:err.message})).setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { const d = e.parameter; const ss = SpreadsheetApp.getActiveSpreadsheet(); if (d.action === 'read' && d.sheet) { const sh = ss.getSheetByName(d.sheet); const rows = sh ? sh.getDataRange().getValues() : []; return ContentService.createTextOutput(JSON.stringify({ok:true,rows})).setMimeType(ContentService.MimeType.JSON); } return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON); }
步驟 3：部署並貼上網址
  1. 在 Apps Script 點「部署 → 新增部署」
  2. 類型選「網路應用程式」
  3. 「執行身分」選「我」，「存取權」選「所有人」
  4. 按「部署」，複製產生的網路應用程式網址
  5. 將網址貼在下方
